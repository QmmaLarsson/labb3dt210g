# Laborationsuppgift 3 för kursen DT210G, Fördjupad frontend-utveckling
**Namn:** Emma Larsson\
**Student-ID:** emla2309

**Syfte med uppgift:**
Syftet med uppgiften var är att utveckla en Single Page Application med React och TypeScript där följande implementerades:
- Routing med React Router
- JWT-baserad autentisering
- Skyddade routes
- CRUD-operationer mot REST API
- Väldefinierade TypeScript-interfaces

**Lösning på uppgift:**
Applikationen är en webbapplikation där användare kan se en publik lista med blogginlägg, visa enskilda blogginlägg (även detta publikt), logga in via JWT-autentisering och hantera innehåll i en skyddad administrationsdel.
